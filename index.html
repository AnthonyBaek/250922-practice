안녕하세요! 👋

안녕하세요! 저는 김개발입니다. 사용자 중심의 웹 서비스를 만드는 것을 좋아하는 개발자입니다. 항상 새로운 기술을 배우고 적용하는 것에 열정을 가지고 있으며, 팀워크를 중시합니다.

경력 & 경험

3년간의 웹 개발 경험을 바탕으로 프론트엔드와 백엔드 개발을 모두 경험했습니다. 특히 React, Vue.js를 활용한 SPA 개발과 Node.js 기반의 서버 개발에 전문성을 가지고 있습니다.

다양한 프로젝트를 통해 사용자 경험 개선과 성능 최적화에 대한 노하우를 쌓았으며, 협업 도구를 활용한 효율적인 개발 프로세스 구축에 관심이 많습니다.

기술 스택

JavaScript React Vue.js Node.js Python HTML/CSS MongoDB Git

성격 & 가치관

긍정적이고 적극적인 성격으로 어려운 문제도 포기하지 않고 해결하려고 노력합니다. 동료들과의 원활한 소통을 중요하게 생각하며, 지속적인 학습을 통해 성장하는 개발자가 되고 싶습니다.

사용자의 입장에서 생각하는 개발을 지향하며, 코드의 가독성과 유지보수성을 고려한 깔끔한 코드 작성을 추구합니다.
